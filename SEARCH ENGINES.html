Search Engine Optimization (SEO)
There are two places where SEO is implemented. First in websites and second in keyword advertising.  Here I will a address the main points regarding SEO and Websites.

Preparation
There is little reason to spend time and money optimizing a website
for search engines if that web site is not presentable. Visitors will not stay if your
website is not:
  • Aesthetically appealing
  • Focused
  • Easy to use
  • Mobile friendly

All of these points reduce credibility and increase the likelihood of a visitor moving on.

Search-ability
Currently there are two key search engines in North America: Google and Bing.
In order for these search engines to be aware of you site it must be indexed. A web site
cannot be indexed if it contains a robots.txt file which contains directives to ignore
a site. For details see...

Keywords
Keywords are the words and terms typed into a search engine. A web site should be designed
to match keyword searches by including words and terms that a search engine can easily use.
The goal of a search engine is to abstract content carefully, so that it can present accurate and
useful results.

What search engines use
Search engines can be told about keywords and given descriptions-but they do their own thinking now.  The "keywords" "meta tag and the "description" meta tag are still used, but they are largely de-emphasised in favor of website content and website name.   In a sense, search engines define your keywords for you.

Credibility is a key issue in how search engines rank results. The most important factor in credibility is backlinks. In other words
the number of other websites that point to your website. No question search engines filter out websites containing foul words too. 

How to structure content for SEO
Search engines parse and analyze content.  This means the domain name should be a clue to site content. 
In fact, interior URLs should also be used to impart information related to search terms.  Next comes the site title which should be  highly supportive of (and consistent with) site content. 

Search engines factor site structure and hierarchy to further quantify content. For this reason, placing an important term in an h1 tag is better than placing it in a div tag with a class to emphasize it.  For the same reason, text with a larger font or placed nearer the top of a site is given more weight in evaluating the purpose of a site.

Conversely, images and embedded content such as video, do not support a site theme because they are not read or interpreted by search engines.   At best, a search engine can using image and video file names to support a sites theme.
